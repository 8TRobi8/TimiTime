## Color Scheme

### Task Urgency Colors
- **Green** (#34c759): Due date is in the future
- **Yellow** (#ff9500): Due today (no flexibility) OR overdue within flexibility window
- **Red** (#ff3b30): Overdue beyond flexibility period

### Theme Colors
- **Tint** (Light): #0a7ea4 (Blue)
- **Tint** (Dark): #ffffff (White)
- **Background** (Light): #ffffff
- **Background** (Dark): #151718
- **Border** (Light): #e5e5ea
- **Border** (Dark): #38383a

## Interactions

### Task Card
- **Tap**: Toggle completion status
- **Visual**: Strikethrough + reduced opacity when completed

### Buttons
- **Primary**: Tint color background, white text
- **Secondary**: Gray background
- **Hover/Active**: Slight opacity change
- **Disabled**: 60% opacity

### Modals
- **Open**: Slide up from bottom
- **Close**: Slide down / tap outside / Cancel button
- **Background**: Semi-transparent overlay

### Navigation
- **Tab Bar**: Bottom of screen
- **Active Tab**: Tint color
- **Inactive Tab**: Gray color
- **Haptic**: Feedback on tab change

## States

### Loading
- Spinner in center of screen
- Buttons show "Loading..." text
- Inputs disabled

### Empty
- Friendly message: "No tasks yet. Tap 'Add' to create one!"
- Centered in list view

### Error
- Alert dialog with error message
- Dismiss button
- Non-blocking

### Filtered
- Blue banner at top: "Showing filtered tasks Â· Tap to show all"
- Tap banner to clear filter
- Returns to full task list
